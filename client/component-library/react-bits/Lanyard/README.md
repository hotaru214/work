# Lanyard registry source

This directory preserves the source generated by:

```text
npx shadcn@latest add @react-bits/Lanyard-TS-CSS
```

It intentionally lives outside `src/` because the registry entry references
`card.glb` and `lanyard.png`, but does not provide those assets. It also needs
the Three.js, React Three Fiber, Drei, Rapier, and MeshLine runtime stack.

Move it into `src/components/` only after supplying the two assets and adding
compatible React 18 versions of those dependencies.
