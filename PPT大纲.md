## 第 5 页 · 用户认证与安全

| 特性 | 实现方式 |
|---|---|
| 注册校验 | 用户名规则 + 密码强度（大小写/数字/符号 ≥3 类） |
| JWT 登录 | Token 含 typ/iat/jti，7 天有效 |
| 401 拦截 | API 层检测 → 自动清除 Token → 跳转登录 |
| 头像上传 | ≤2MB + 图片格式白名单 |
| 资源权限 | 所有接口校验用户所有权 |

## 第 8 页 · 资料检索 — RAG 核心技术

```
用户提问 → 定位课程 → 查询资料表 → pdfplumber 提取全文
→ 分段（按双换行）→ TF 关键词打分 → Top-K 段落
→ 拼入 System Prompt → DeepSeek 生成回答
```

**支持格式**：.txt .md .py .js .html .json .csv .sql .pdf 等 30+ 种  
**PDF 引擎**：pdfplumber（中文兼容性好）

> 📸 **截图**：
> - 不需要前端截图。此处适合画流程图（PPT SmartArt 或手绘）。
> - 可附上 `retrieval.py` 代码片段的关键几行。
